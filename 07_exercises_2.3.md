## Using given or graphed limits (Exercises 1–2)

**1.** Given that
$$\lim_{x \to 2} f(x) = 4, \qquad \lim_{x \to 2} g(x) = -2, \qquad \lim_{x \to 2} h(x) = 0$$
find the limits that exist. If the limit does not exist, explain why.
- (a) $\displaystyle \lim_{x \to 2}[f(x) + 5g(x)]$
- (b) $\displaystyle \lim_{x \to 2}[g(x)]^3$
- (c) $\displaystyle \lim_{x \to 2}\sqrt{f(x)}$
- (d) $\displaystyle \lim_{x \to 2}\frac{3f(x)}{g(x)}$
- (e) $\displaystyle \lim_{x \to 2}\frac{g(x)}{h(x)}$
- (f) $\displaystyle \lim_{x \to 2}\frac{g(x)h(x)}{f(x)}$

**2.** The graphs of $f$ and $g$ are given. Use them to evaluate each limit, if it exists. If the limit does not exist, explain why.
- (a) $\displaystyle \lim_{x \to 2}[f(x) + g(x)]$
- (b) $\displaystyle \lim_{x \to 1}[f(x) + g(x)]$
- (c) $\displaystyle \lim_{x \to 0}[f(x)g(x)]$
- (d) $\displaystyle \lim_{x \to -1}\frac{f(x)}{g(x)}$
- (e) $\displaystyle \lim_{x \to 2}[x^3 f(x)]$
- (f) $\displaystyle \lim_{x \to 1}\sqrt{3 + f(x)}$

> **Graphs provided.** Two curves $y = f(x)$ and $y = g(x)$ on separate axes.

## Priority gradient

**Highest value — the core techniques:**

- **Exercises 11, 15, 18** — factor-and-cancel $(0/0)$ with simple polynomials. The basic mechanic.
- **Exercises 17, 20, 28** — **difference quotients** of the form $((a + h)^n - a^n)/h$ or $((a+h)^{-1} - a^{-1})/h$. Answer is $n a^{n-1}$ or $-a^{-2}$. **These are Chapter 3 derivatives in disguise.** Must-do.
- **Exercises 21, 22, 23, 27, 30** — rationalization. Worth doing a few to build the reflex. Exercise 22 ($\sqrt{1+h} - 1)/h = 1/2$ is the derivative of $\sqrt{x}$ at 1 — foreshadowing.
- **Exercises 35, 36, 37** — Squeeze Theorem with explicit bounds. Useful to cement the technique.
- **Exercises 47, 48** — piecewise / absolute value limits. Standard test-question flavor.
- **Exercises 55, 56** — **reverse engineering**. Given a limit value, deduce related quantities. The technique: recognize $(f(x) - f(a))/(x - a) \to L$ means $f(a) = $ whatever makes the numerator → 0, and $L = f'(a)$. Exercise 55 has $f(1) = 8$ and $f'(1) = 10$; Exercise 56 forces $f(0) = 0$ and $f'(0) = 0$ (the $f/x^2$ form requires both). **Direct-translate test for Chapter 3's derivative-from-limit problems.**
- **Exercise 61** — find the value of $a$ making a limit exist. Classic "clear the $0/0$" technique: since denominator $\to 0$ at $x = -2$, numerator must also $\to 0$, giving $3(-2)^2 + a(-2) + a + 3 = 0 \Rightarrow 12 - 2a + a + 3 = 0 \Rightarrow a = 15$. Then the limit is computed by factoring.
- **Exercise 62** — geometric limit problem. $Q = $ upper intersection. Setup, write an equation, parameterize $r$, and find the limit $R = (4, 0)$ as $r \to 0^+$. Neat synthesis.

**High value:**

- **Exercises 3–9** — basic direct-substitution practice (DSP); pick 2–3.
- **Exercises 39, 40, 42** — absolute value + one-sided; pick 2.
- **Exercise 45** — sign function. The signature of every trading strategy / perceptron / piecewise indicator. Do it.
- **Exercise 52** — $L \to 0$ as $v \to c^-$: the relativistic length contraction going to zero at the speed of light. Physics connection.

**Medium value:**

- **Exercises 1, 2** (given-limits / graph-reading) — pick one.
- **Exercises 10, 12, 13, 14, 16, 19, 24, 25, 26, 29** (more factoring / rational) — just drills; pick 3.
- **Exercises 31, 32** (graphing + confirm analytically) — pick one.
- **Exercises 33, 34** (Squeeze with graphing) — pick one.
- **Exercises 46, 49, 50** (floor, piecewise with floor) — pick one.

**Skim or skip:**

- **Exercises 53, 54** (prove DSP for polynomials / rationals) — the proof uses repeated Law application; mechanical.
- **Exercises 57, 58, 59, 60** — clever but non-essential.
- **Exercise 51** (floor sum identity) — niche.

**Most important exercises for Chapter 3 prep:** **17, 20, 22, 28, 55, 56.** These are difference-quotient calculations that will resurface immediately when Chapter 3 defines the derivative as $\lim_{h \to 0}(f(a+h) - f(a))/h$. Getting fluent with these now makes the Chapter 3 transition trivial.

**Exercise 61 is a little gem** — whenever you see a $P(x)/Q(x)$ with a parameter and the question "for what value does the limit exist at $a$?", the answer is always: set $P(a) = 0$ (force cancellation with $(x - a)$), solve for the parameter, then compute the limit by factoring. Same technique is used for making rational expressions *continuous* (Section 2.5), determining parameters in Taylor/Padé approximations, and cleaning up removable singularities in contour integration.
